依赖与致谢 ​

更新: 6/25/2025 字数: 0 字 时长: 0 分钟

本文档的撰写和构建得益于以下优秀的开源项目和社区资源。我们在此向所有贡献者致以诚挚的感谢！

开源项目 ​

  • VitePress: 提供了强大的静态站点生成能力，为本文档的搭建奠定了基础。
  • Vue: 作为核心前端框架，Vue 赋予了本文档交互性和动态性。
  • Element Plus: 提供了丰富的 UI 组件，助力本文档实现了美观的用户界面。
  • Naive UI: 另一个优秀的 UI 组件库，为本文档的设计提供了更多可能性。

文档样式参考 ​

在排版和布局上，本文档借鉴了以下开源项目的官方文档样式，力求提供清晰、友好的阅读体验：

  • Vue.js 官方文档: 简洁明了的结构和出色的内容组织。
  • VitePress 官方文档: 优雅的布局和高效的信息呈现方式。
  • VuePress Theme Hope: 提供了丰富的文档主题功能和美学设计。
  • vuepress-reco: 另一个优秀的 VuePress 主题，提供了多种布局和组件。
  • 原神地图: 独特的交互式地图展示和数据可视化。
  • UnoCSS: 学习了其原子化 CSS 的应用和排版优化思路。
  • VitePress 快速上手: 提供了 VitePress 文档构建的实用案例和技巧。
